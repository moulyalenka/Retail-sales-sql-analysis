# Retail Sales SQL Analysis

## Project Overview
This project analyzes retail sales data using SQL to derive insights into sales performance, customer behavior, and product trends.

## Database Schema
- Customers
- Categories
- Products
- Orders
- Order_Items
- Payments

## Key Analysis Areas
### 1. Sales Analysis
- Total revenue trends
- Monthly sales growth
- Top-performing categories

### 2. Customer Insights
- Customer lifetime value
- Repeat vs one-time customers
- Purchase frequency

### 3. Product Performance
- Best-selling products
- Low-performing products
- Category-wise contribution

## Tools Used
- MySQL
- DB Fiddle / Terminal
- Git & GitHub
